Open Parking and Camera Violations

This webpage will allow you to generate cards based on the data from Open Parking and Camera Violations.

INSTRUCTIONS

Input only one search option at a time, otherwise it will only pick one. In order to use these input a specific type of violation, plate number, or state abbreviation in caps depending on the type of data you want to search by. Once done, click the Search button in order to generate cards detailing data such as the violation, which state it happened in, the plate number and the amount fined. If an image exists, you may click a card in order to view the full details of the violation report through a modal.
