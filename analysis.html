Open Parking and Camera Violations

Select Chart Type

This webpage will allow users to select one of two types of graphs for the data from NYC Open Data, a Bar or Pie graph. Users may graph data based on the type of violation, how many violations each state has, and the agencies that issued these violations. The bar graph will show the amount of each type of data there is, and the bar graph will display percentages of the data to see what makes it up.
